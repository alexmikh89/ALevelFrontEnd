Блоковый тепловой пункт

Блоковий индивидулаьный тепловой пункт - это штука вместо бойлера для отопления помещений.

БИТПы бывают:

Промышленного назначения

  • Для отопления огромных ангаров космических масштабов с кучей сквозняков

  • Для поддержания теплоты и уюта в маленьких подсобочках, чтоб было тепленько спать в перерыв

Общенародного назначения

  • Для отопления больших домиков

  • Для отопления средненьких домиков

  • Для отопления маленьких домиков

  • Для отопления квартирок

bitp consists of

  1. Pipes
  2. Valves
  3. Filters

Сборка

БИТП собирается грязными слесарями, сварщиками и кувалдой

Скорость постройки

Скорость постройки варьируется в широких приделах и оценивается относительным коэффициентом качества ♺
